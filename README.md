# Code Generator

# Описание

Консольное приложение на C++ для генерации и проверки уникальных 12-значных кодов по заданным правилам.

# Правила генерации

- Третий разряд ≤ 5
- Десятый разряд — чётный
- Сумма третьего и пятого разрядов > 9
- Первый разряд - цифровой корень суммы остальных цифр
- Код делится на 7 и 13, но не делится на 2

# Возможности

- Генерация кодов
- Проверка валидности
- Определение повторного использования (активации)
- Определение локально сгенерированных кодов

# Назначение

Имитация системы лицензионных или активационных кодов.
