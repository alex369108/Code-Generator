# Code-generator

# Программа создаёт 12-значный код по правилам:
- Третий разряд ≤ 5.
- Десятый разряд – чётный.
- Сумма третьего и пятого разрядов > 9.
- Первая цифра – цифровой корень суммы остальных 11 цифр.
- Код должен делиться на 7 и на 13, но не делиться на 2.

Также программа проверяет валидность кода, активировался он ранее и был ли он создан на этом устройстве
